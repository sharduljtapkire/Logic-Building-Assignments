This directory contains five simple C programs focused on basic loops, patterns, and arithmetic operations. Each file includes a main function and one additional function to perform a specific task based on user input.

1. Question1.c
Purpose:
Prints a pattern of $ * repeated iNo times.

Function: void Pattern(int iNo)

Example Input:
5

Output:
$ * $ * $ * $ * $ *

2. Question2.c
Purpose:
Displays numbers from 1 to the entered number.

Function: void Display(int iNo)

Example Input:
4

Output:
1 2 3 4

3. Question3.c
Purpose:
Displays numbers from -iNo to iNo.

Function: void Display(int iNo)

Example Input:
3

Output:
-3 -2 -1 0 1 2 3

4. Question4.c
Purpose:
Displays all odd numbers from 1 to the entered number.

Function: void OddDisplay(int iNo)

Example Input:
10

Output:
1 3 5 7 9

5. Question5.c
Purpose:
Displays the first five multiples of the entered number.

Function: int MultipleDisplay(int iNo)

Example Input:
4

Output:
4 8 12 16 20
